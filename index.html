About the Project

Introduction

PSLFjobs.com is a live and running website, offering subscription-based PSLF
Eligibility Assessment to Enterprise and Individual borrowers.

PSLF stands for Public Service Loan Forgiveness

Goal

The main goal of this platform is to provide PSLF support for all the eligible candidates who
are working in either a Government Organization or a Non-Profit Organization based on a
monthly or yearly subscription

Requirements fulfilled

UI / UX Strategy and Design (Main Dashboards)

  • Create and implement UX and UI strategy and designs for a web app and website to
    facilitate easy to use, elegant and minimalist design that fits the industry standards.
  • Follow the user journey based on buyer personas.
  • Easy to use - user should be able to perform a given activity with
    an optimal number of touchpoints/interactions
  • Must carry PSLF branding
    1. Brand Color: #DF8E06
    2. SVG Logo and wordmark will be sent separately. Bull’s eye is the Brand Logo, and the
      following is the wordmark.
    3. Highlight items yellow/red lights that need attention or use a similar strategy.
PSLFjobs Widget
  • PSLF Widget is placed on different partner websites. We should be able to catch and
    analyse data and recommendations from widget in a single place in the web application.

Types of Users and Workflows

This project has 3 major types of Users who has dashboards

  • HR Admin
  • Enterprise Employee
  • Individual Borrower
HR Admin
  • The HR Admin takes care of all the documentation needs to be done for all the
    Enterprise Employees under him.
  • The Roles HR Admin and Enterprise Employee belongs in a single company.
Enterprise Employee
  • One of the Employee who works together with others and links the
    documentation to HR Admin for verification.
  • Documentation is related to PSLF verification.
Individual Borrower
  • The solo person who doesn’t belong to a company but likes to check his PSLF and
    verify documentation on his own

Live version of the product

  • The live version of the product dashboard screens can be seen by visiting the
    website https://benelevate.com and signing up.
  • The PSLF widget screens are accessible for everyone even without an account.
  • The current web traffic for the website is more than 15k visits/day
